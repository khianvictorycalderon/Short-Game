# Red vs Blue Square Game

This is a simple two-player web game built using **vanilla HTML, CSS, and JavaScript**. No frameworks are required.  

## How to Play

- **Red Square (Player 1)**
  - Move using **W, A, S, D** keys.
- **Blue Square (Player 2)**
  - Move using **Arrow keys**.

The goal is for the red square to collide with the blue square before the 30-second timer runs out.  

- If **Red** collides with **Blue**, **Red wins!**
- If the **timer runs out** before collision, **Blue wins!**

## Features

- Real-time movement for two players.
- Collision detection.
- Countdown timer display.
- Responsive to browser window size.

## How to Run

1. Clone or download the project files.
2. Open `index.html` in your browser.
3. Start pressing keys to move the squares and start the game timer automatically.
